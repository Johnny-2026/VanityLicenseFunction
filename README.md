# MoonJumpServer(VanityLicense)

**MoonJumpServer-VanityLicense** 是一个基于 Flask 的 Minecraft 云插件分发与服务器管理系统。配套 C++ 原生客户端通过 JNI 实现内存级插件注入，不落盘运行，配合 Bukkit/Paper 服务端使用。

> **项目定位**：为 Minecraft 服务器网络提供中心化的插件分发、实例注册、心跳保活、远程重启等云管理能力。

## 功能特性

### 核心能力
- **插件加密分发** — 上传 .jar 后自动用 AES-256-GCM 加密存储，下载时用会话密钥重加密传输
- **一次消费制** — 每个会话对每个插件只能下载一次，防泄露
- **IP 绑定认证** — 动态 Token 注册后绑定 IP，HMAC-SHA256 签名 + 时间戳防重放
- **心跳保活** — 客户端每 4 秒发送心跳，超时 350 秒自动清理离线实例
- **远程重启** — 管理员可重启指定服务器/IP/全部实例，通过心跳响应下发重启指令

### 管理面板
- **插件管理** — 上传、删除、查看加密插件元数据（自动解析 plugin.yml）
- **服务器管理** — 添加/编辑/删除服务器绑定，配置每个服务器允许的插件列表
- **Token 管理** — 查看所有注册实例、在线/离线状态、批量删除离线 Token
- **2FA 支持** — 基于 TOTP 的双因素认证，可选启用
- **人机验证** — PoW（工作量证明）+ Cloudflare Turnstile 双重防护
- **中国 IP 优化** — 自动识别中国 IP 跳过 PoW，提升访问体验
- **实时统计** — 插件数、绑定服务器数、在线实例数，每 5 秒自动刷新

### 安全特性
| 特性 | 说明 |
|---|---|
| 速率限制 | 滑动窗口，每端点独立配置 |
| 突发检测 | 10s 内 50 个不同 IP 触发强力防护模式 |
| PoW 工作量证明 | SHA-256 14 位前导零（约 16K 哈希），60s 有效 |
| Turnstile | Cloudflare 验证码 |
| 会话文件短时效 | 30 秒过期自动清理 |
| 路径穿越防护 | session_id 仅允许十六进制字符 |
| 反调试（客户端） | Windows `IsDebuggerPresent` / Linux `ptrace` + `TracerPid` |
| 自定义 DNS（客户端） | 强制通过 `1.1.1.1` 解析，绕过系统 DNS |

## 技术栈

| 层 | 技术 |
|---|---|
| 后端框架 | Python Flask |
| 数据库 | SQLite |
| 加密 | AES-256-GCM (cryptography / OpenSSL) |
| 管理面板 | Jinja2 模板 + 原生 CSS + JavaScript |
| 认证 | HMAC-SHA256, TOTP (pyotp), Cloudflare Turnstile |
| 客户端 C++ | WinHTTP / libcurl, BCrypt / OpenSSL, JNI |
| 客户端 Java | Bukkit/Paper API (1.21.x) |
