# VanityLicense

> **Minecraft 服务端云插件管理系统**

VanityLicense 是一套完整的云插件管理解决方案，包含 Web 管理后台、Bukkit/Spigot 客户端插件和原生高性能层。支持插件加密上传、云端分发、内存注入和服务器全生命周期管理。

---

## 功能特性

### 管理后台
- 插件上传、加密存储、版本管理
- 服务器绑定与插件授权管理
- 动态 Token 管理，在线/离线状态实时监控
- 单实例/批量远程重启
- 安全事件报告面板
- 双因素认证 (TOTP) + 人机验证 (PoW)

### 客户端
- 零磁盘插件注入，纯内存加载
- 硬件指纹绑定 + 动态 Token 认证
- 4 秒心跳保活，支持远程重启指令
- 多层级反调试/反转储保护
- 跨平台支持 (Linux / Windows)

## 安全特性

| 防护层 | 措施 |
|---|---|
| **传输** | HTTPS + 请求签名 + 时间戳防重放 |
| **存储** | AES-GCM 加密 + 会话一次性消费 |
| **认证** | 硬件绑定 + 动态 Token + IP 锁定 + TOTP |
| **访问** | PoW 人机验证 + 登录限速 + 突发流量检测 |
| **运行时** | 反调试 / 反转储 / 反 Agent 注入 / Watchdog |
